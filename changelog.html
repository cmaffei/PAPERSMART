programme.html, line ~221

Problem: When viewing session page in landscape mode on ipad, a left arrow appears to the left of the Programme link and the P is truncated from the word programme

There is a inline script before the </body>: $( '#carousel' ).elastislide({start : 3, minItems: 2}); and you have to change the start to show:
1 - PROGRAMME;
2 - AGENDA;
3 - DOCUMENTS;
4 - STATEMENTS.
Updating this script with right start will solve the problem.
